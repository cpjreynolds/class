### Example 7

Show that `lim[x->0](|x|) = 0`

`|x| = { x : x > 0; -x : x < 0 }`

`lim-[x->0](|x|) = lim-[x->0](-x) = 0`

`lim+[x->0](|x|) = lim+[x->0](x) = 0`

Since `lim-[x->0](|x|) = lim+[x->0](|x|) = 0`: `lim[x->0](|x|) = 0`

### Example 8

Prove that `lim[x->0](|x| / x)` does not exist.

```
lim[x->0](|x| / x)

    = lim[x->0]({ x / x : x > 0; -x / x : x < 0 })

    = lim[x->0]({ 1 : x > 0; -1 : x < 0 })

    = lim-[x->0](|x| / x) = -1

    = lim+[x->0](|x| / x) = 1
```

Since `lim-[x->0](|x| / x) != lim+[x->0](|x| / x)`: `lim[x->0](|x| / x) = DNE`.
